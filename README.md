## ðŸš¦ Run

To get full experience, you must run web and mobile environment.

1. Run back-end:

```bash

#nlw-copa/server

npm install && npm run dev

#or

yarn && yarn dev

```

2. Run web version:

```bash

#nlw-copa/web

npm install && npm run dev

#or

yarn && yarn dev

```

3. Run web version:

```bash

#nlw-copa/mobile

npm install && npm run start

#or

yarn && yarn start

```
